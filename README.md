# Graphic Design Portfolio

A Next.js portfolio for graphic design, video editing, motion, and campaign work.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the local development server:

```bash
npm run dev
```

Open `http://localhost:3000`.

## Editing Content

Most portfolio content lives in:

```txt
src/data/portfolio.ts
```

Replace the placeholder profile, project text, image URLs, video embeds, tools, and deliverables with your real work.

## Deployment

This project is ready for Vercel. Connect the GitHub repository in Vercel, keep the framework preset as Next.js, and deploy.
