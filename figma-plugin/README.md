# Nour Portfolio Website Boards Figma Plugin

This local Figma plugin creates editable design boards for:

- Royal Dagish
- IGS MarketPlace

It generates Figma pages with:

- Cover frames
- Desktop landing structure
- Mobile responsive screens
- Component/library notes
- Repo and stack references

## Load In Figma Desktop

1. Open Figma Desktop.
2. Create or open a blank design file.
3. Go to `Plugins > Development > Import plugin from manifest...`.
4. Select:

```txt
C:\Users\nour_\Documents\Graphic portfolio\figma-plugin\manifest.json
```

5. Run `Plugins > Development > Nour Portfolio Website Boards`.

The plugin will create two new pages in the current Figma file.

## Notes

This creates editable Figma structure, components, typography, and layout placeholders. It does not import the full web app pixel-for-pixel or embed local screenshot PNGs automatically. After running it, you can drag the preview images from:

```txt
C:\Users\nour_\Documents\Graphic portfolio\public\project-previews
```

into the matching image placeholders if you want screenshot-backed mockups.
